Vike extensions integrate tools, so that you don't have to manually integrate them yourself.

They are optional. You can use Vike without any Vike extension and have complete control over integration code instead, see <Link href="/why#optional-control" />.

## React

 - <Link href="/vike-react">`vike-react`</Link>
 - [`vike-react-query`](https://github.com/vikejs/vike-react/tree/main/packages/vike-react-query)
 - :construction: [`vike-react-zustand`](https://github.com/vikejs/vike-react/pull/57)

> Contribution welcome to write extensions.
> - [#1373 - Vike Extensions: GraphQL](https://github.com/vikejs/vike/issues/1373)
> - [#1374 - Vike Extensions: Data Management Stores (Redux, Vuex, ...)](https://github.com/vikejs/vike/issues/1374).
