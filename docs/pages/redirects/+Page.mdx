Permanent redirections (HTTP status code `301`):

```js
// pages/+config.js

export default {
  redirects: {
    '/about-us': '/about',
    // Parameterized redirections
    '/product/@id': '/buy/@id',
    // Glob redirections
    '/admin/*': '/private/*',
    // External redirections
    '/admin/*': 'https://admin.example.org/*'
  }
}
```
