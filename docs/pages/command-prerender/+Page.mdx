Vike CLI command to trigger <Link href="/pre-rendering">pre-rendering</Link>.

```bash
$ vike prerender
```

> Use <Link href="/prerender#disableautorun" /> if you don't want pre-rendering to be run twice.

The Vike CLI doesn't have any option, instead:
 - Use the <Link href="/prerender">`prerender` setting</Link>, or
 - use the <Link href="/prerender-programmatic">`prerender()` API</Link>.
