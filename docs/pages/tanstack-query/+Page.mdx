You can use Vike with [TanStack Query](https://tanstack.com/query).

This page documents how to manually integrate TanStack Query.

> Instead of manually integrating TanStack Query yourself, you can use:
> - [`vike-react-query`](https://github.com/vikejs/vike-react/tree/main/packages/vike-react-query)
> - [`vike-vue-query`](https://github.com/vikejs/vike-vue/tree/main/packages/vike-vue-query)
