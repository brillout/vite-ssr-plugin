We recommend using [Bati](https://batijs.dev) for creating new apps.

> Apps scaffolded by Bati use <UiFrameworkExtension name noLink links/>.

## Without `vike-{react,vue,solid}`

You can also create a new Vike app that doesn't use <UiFrameworkExtension name noLink links/>, which means you control the React/Vue/Solid integration.

<Warning>
Do this only if you are an expert user and only if you have a strong & clear reason for not using <UiFrameworkExtension name noLink />.
</Warning>


With [npm](https://www.npmjs.com):

```shell
npm create vike@latest
```

With [pnpm](https://pnpm.io):

```shell
pnpm create vike
```

With [Bun](https://bun.sh):

```shell
bun create vike
```

With [Yarn](https://yarnpkg.com):

```shell
yarn create vike
```

Options:
- `--skip-git`: don't initialize a new Git repository

A prompt will let you choose between:
- `react`: React + JavaScript
- `react-ts`: React + TypeScript
- `vue`: Vue + JavaScript
- `vue-ts`: Vue + TypeScript
